📡 API Documentation

API פתוח לקבלת נתוני מערכת השעות של המכללה הטכנולוגית של חיל האוויר, באר שבע

Base URL: https://timetable-bistbash.vercel.app/api

כל ה-endpoints מחזירים JSON, תומכים ב-CORS, ומשתמשים ב-cache של 2 דקות.

GET /api/classes

רשימת כל הכיתות במכללה

GET /api/classes

Response:
{
  "classes": [
    {"id": "1", "name": "ט - 1"},
    {"id": "57", "name": "י - 1 תוכנה"},
    ...
  ],
  "total": 40
}

זמני השיעורים (12 שעות ביום)

GET /api/hours

    {"index": 1, "start": "08:15", "end": "09:00"},

GET /api/timetable

מערכת השעות של כיתה מסוימת

Parameters

שםסוגברירת מחדלתיאור
classIdstring1מזהה כיתה (מ-/api/classes)
viewstringTimeTableסוג תצוגה (ראה למטה)
weekinteger0הסטת שבוע: 0=השבוע, 1=שבוע הבא, -1=שבוע שעבר
flushany—אם קיים, עוקף cache

Views

ערךתיאורסוג תגובה
TimeTableמערכת קבועהGrid (days + lessons)
ChangesTableמערכת ושינויים (כולל מבחנים ואירועים)Grid (days + lessons)
Changesרשימת שינוייםList (changes)
ExamsמבחניםList (changes)
Messagesהודעות כלליותList (changes)
EventsאירועיםList (changes)

Grid Response (TimeTable / ChangesTable)

GET /api/timetable?classId=1&view=TimeTable

Response:
{
  "days": [
    {
      "name": "יום ראשון",
      "date": "22.03",
      "isToday": true,
      "lessons": [
        "מסדר בוקר\nראובן יוחנן",
        "עברית (207 - מעבדת מחשבים)\nטיביב בוידיקמן מיריאנה",
        ...
      ]
    },
    ...
  ],
  "hours": 12,
  "changes": [],
  "updateTime": "20.03.2026 13:41",
  "week": 0
}

מבנה lessons: כל תא הוא string עם שורות מופרדות ב-\n. בהקבצות, כל הקבצה מורכבת משורת מקצוע (עם חדר בסוגריים) ואחריה שורת מורה.

סימני שינויים:

קידומתמשמעות
[ביטול]שיעור מבוטל
[החלפה]מילוי מקום — שורה ראשונה: מקצוע, שורה שנייה: "מחליף: שם"
[מבחן]מבחן (רק ב-ChangesTable)
[אירוע]אירוע (רק ב-ChangesTable)

GET /api/lesson מומלץ

API מורחב — מחזיר שיעורים מפורסרים עם סינון מתקדם. כל שיעור מוחזר כאובייקט מובנה עם שדות נפרדים למקצוע, מורה, חדר וסוג.

Parameters

שםסוגברירת מחדלתיאור
classIdstring1מזהה כיתה
viewstringTimeTableסוג תצוגה (TimeTable / ChangesTable / Changes / Exams / Events)
weekinteger0הסטת שבוע
dayinteger—סנן יום: 0=ראשון, 1=שני ... 5=שישי
hourinteger—סנן שעה: 0-11
fieldsstring—שדות לכלול בתגובה (מופרד בפסיקים): subject,teacher,room,type,day,hour
searchstring—חיפוש חופשי בשם מקצוע, מורה או חדר

דוגמאות

# מה יש ביום שני שעה 3?
GET /api/lesson?classId=1&day=1&hour=3

# רק שמות מקצועות ומורים
GET /api/lesson?classId=1&fields=subject,teacher,hour,dayName

# חפש את כל השיעורים של מורה מסוימת
GET /api/lesson?classId=1&search=כהן שילת

# חפש מתמטיקה
GET /api/lesson?classId=1&search=מתמטיקה

# מערכת עם שינויים, רק ביטולים
GET /api/lesson?classId=1&view=ChangesTable&search=ביטול

# שעה 5 בכל הימים, רק מקצוע וחדר
GET /api/lesson?classId=1&hour=5&fields=dayName,subject,room

דוגמאות שימוש

JavaScript (fetch)

// קבלת מערכת השעות של ט-1 עם שינויים
const res = await fetch('https://timetable-bistbash.vercel.app/api/timetable?classId=1&view=ChangesTable');
const data = await res.json();

data.days.forEach(day => {
  console.log(day.name, day.date);
  day.lessons.forEach((lesson, hour) => {
    if (lesson) console.log(`  שעה ${hour}: ${lesson.split('\n')[0]}`);
  });
});

Python

import requests

# רשימת כיתות
classes = requests.get('https://timetable-bistbash.vercel.app/api/classes').json()
for c in classes['classes']:
    print(f"{c['id']}: {c['name']}")

# מערכת שעות
data = requests.get('https://timetable-bistbash.vercel.app/api/timetable', 
    params={'classId': '1', 'view': 'TimeTable'}).json()
for day in data['days']:
    print(f"\n{day['name']} {day['date']}")
    for i, lesson in enumerate(day['lessons']):
        if lesson:
            print(f"  שעה {i}: {lesson.split(chr(10))[0]}")

הגבלות

ה-API מבוסס על scraping מאתר שחף ולכן:

Cache2 דקות — תוצאות נשמרות כדי לא להעמיס על שחף
Timeout30 שניות — שבועות רחוקים (3+) עלולים להיות איטיים
Rate limitאין הגבלה רשמית, אך שימוש מוגזם יכול לגרום לחסימה משחף
